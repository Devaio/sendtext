Sendtext

Send text messages from your terminal for free

Installation

npm install -g sendtext

Usage

From the terminal:
  • $ sendtext -m
  • Number: (AreaCode)Number
  • Message: Add Message

Example

  • Number: 1230983456
  • Message: Hello World

Notes

  • No spaces between the AreaCode and the Number
  • Number: Maximum of 10 Numbers
